# 🍗 大鸡腿动量轮动交易系统 (Momentum Rotation System)

> “股市最大悖论之一：看似过高的通常会更高，看似过低的通常会更低” —— 威廉・欧奈尔

这是一个基于 Python 和 Streamlit 构建的**反人性量化交易看板**。本系统通过“动量优胜劣汰”与“目标波动率风控”双轮驱动，旨在帮助交易者克服“追涨杀跌、盲目抄底、死扛回撤”的人性弱点，将不确定的市场变成确定的执行计划。

## 🌟 核心亮点
- **穿越牛熊的战绩**：过去 14 年回测数据中，13 年正收益，仅 1 年微亏（-0.6%）。
- **优胜劣汰机制**：内置黄金、纳指、创业板、沪深300等低相关性资产，系统每日自动计算收益斜率与稳定性，谁强买谁。
- **机构级风控 (目标波动率)**：市场恐慌、波动率飙升时自动降低仓位，保住利润；平稳时满仓进攻。
- **硬核评价体系**：全面展示夏普比率、索提诺比率、最大回撤等专业指标，追求极致的“收益性价比”。
